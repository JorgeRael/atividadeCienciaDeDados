Um exercício de ciência de dados.

O objetivo é produzir um mapa do Ceará com uma indicação visual da taxa de crimes violentos letais e intencionais (CVLI) para cada cem mil habitantes de todas as cidades.

Eu fiz o exercício de tal forma que não é necessário baixar os arquivos, nem fazer tratamento de dados com uma ferramenta externa (como o Excel). O próprio código tem instruções para estas tarefas.
